IO_ISOBMFF
======

ISOBMFF binary perser dumper, converter, powered by PHP.
bidirectional converter with ISOBMFF

# install

```
% composer require yoya/io_isobmff
```

# require

- IO_Bit
 - https://github.com/yoya/IO_Bit
- IO_ICC
 - https://github.com/yoya/IO_ICC
